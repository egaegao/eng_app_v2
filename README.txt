# Weekly Report Dummy Project

Isi folder:
- data/: file CSV dummy
- scripts/weekly_analysis.py: script analisa awal
- output/: hasil summary dan chart setelah script dijalankan
- requirements.txt: library yang dibutuhkan

Cara jalan:
1. Buka folder ini di VS Code
2. Install library:
   pip install -r requirements.txt
3. Jalankan:
   python scripts/weekly_analysis.py
